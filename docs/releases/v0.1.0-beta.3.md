# Time Series Explorer v0.1.0-beta.3

Beta of the Full Desktop edition for Windows 10/11 x64. It supersedes beta.2 and is built with the supported Node.js 22 release environment.

## Downloads

- **Setup:** recommended for most users; installs shortcuts and supports choosing the destination folder.
- **Portable:** runs directly without installation.
- **SHA256SUMS.txt:** hashes for verifying both downloads.

The executables are currently unsigned. Windows SmartScreen may therefore show an "Unknown publisher" warning. Verify that the file was downloaded from this GitHub repository and compare its SHA-256 hash before running it.

## New since beta.2

- **2D Curve Fit:** per-pair linear/quadratic OLS fitting in 2D mode, with a temporal pane, All/Selection range control, CSV export of fit results, help modals, and lazy DuckDB support for large files.
- **Temporal profiles for lazy files:** daily/period temporal-profile analysis now works with lazy-loaded sources, with responsive progress and side-panel warnings.
- **MATLAB datetime/duration:** standalone MATLAB `datetime`/`duration` fields load as time axes, including grouped MCOS datetime fields inside structs/cells.
- **netCDF:** improved generic netCDF time-axis detection.
- **Polish:** CSV parsing tweaks, generated-time autoscale, time-series Y-axis autoscale after axis moves, WebGL for 2D point displays, and analysis range/legend UI refinements.

## Build provenance

- The Menu > Version entry now shows the build's short commit hash and build date, so testers can report exactly which build they are running (web and desktop).

## Desktop capabilities

- Fully local, offline application and bundled processing libraries.
- External documentation links open in the operating system's default browser.
- No classic Electron menu or secondary Electron browser windows.
- Native local-file access and Live Update.
- Higher loading limits and support for large CSV/TXT and Parquet data.
- MATLAB MAT v4 through v7.3 import, including selective matrix import and independent indexes.
- CSV/TXT, Parquet, Excel, generic and PyPSA netCDF and supported pandas Pickle files.
- Time plots, phase plots, 2D curve fitting, state animation, FFT, histograms, correlation, heatmaps and data tools.
- View/session saving and complete-project ZIP saving for in-memory files.

## Requirements and limitations

- Windows 10 or Windows 11, x64.
- No Internet connection is required after downloading.
- Complete-project ZIP export is intentionally unavailable while any loaded source is using lazy loading.
- Automatic updates are not included in this beta. Download newer releases manually from GitHub.
- macOS, Linux and Windows ARM builds are not part of this release.

Please report reproducible problems in the [GitHub issue tracker](https://github.com/ferrucci-franco/timeseries-explorer/issues).
