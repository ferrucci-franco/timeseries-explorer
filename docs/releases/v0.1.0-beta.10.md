# Time Series Explorer v0.1.0-beta.10

Beta release of the Full Desktop edition for Windows, macOS and Linux.

## Downloads

- **Windows 10/11 x64:** installer and portable EXE.
- **macOS Intel:** DMG installer and ZIP package.
- **macOS Apple silicon:** DMG installer and ZIP package.
- **Linux x86_64:** DEB installer and AppImage package.
- **SHA256SUMS files:** checksums for verifying the downloaded packages.

The executables are currently unsigned. Windows SmartScreen may therefore show an **Unknown publisher** warning. Verify that the file was downloaded from the official Time Series Explorer download button and compare its SHA-256 checksum before running it.

## New since beta.9

- **Unified time axis, mixed overlays.** Files with different time formats can now share one plot. Each file's per-file gear opens a **Time axis** menu where you choose its X axis: use the file's own time vector or create a row index, and show it as a calendar, a duration (hh:mm:ss), plain seconds, or a real calendar built from a start date.
- **Promote numeric time.** A simulation whose time is plain seconds can be shown as a duration or promoted to an absolute calendar by fixing its start date, so it overlays real datetime recordings. Duration-vs-seconds is resolved consistently across a panel, independent of trace order.
- **Reindex safeguards.** A warning explains that a generated row index assumes equidistant samples and hides gaps, and the overlay guard blocks — with a clearer, illustrated message — combinations that would misplace traces.
- **Per-axis auto-scale.** New **Fit X** and **Fit Y** toolbar buttons in time-series and 2D modes: fit X to all data, or rescale Y to the data visible in the current X window.
- **Numeric elapsed import.** The CSV import dialog can read a numeric time column as elapsed seconds instead of anchoring it to a datetime.
- **Fixes and polish.** The measurement cursor now lands on the curve when it belongs to the right-hand Y axis; the active file's time-axis menu is tinted so it stands out; a new **Align and reindex files** help section walks through combining very different files; full French, Spanish and Italian translations.

## Desktop capabilities

- Fully local, offline application and bundled processing libraries.
- External documentation links open in the operating system's default browser.
- No classic Electron menu or secondary Electron browser windows.
- Native local-file access and Live Update.
- Higher loading limits and support for large CSV/TXT and Parquet data.
- MATLAB MAT v4 through v7.3 import, including selective matrix import and independent indexes.
- CSV/TXT, Parquet, Excel, generic and PyPSA netCDF and supported pandas Pickle files.
- Time plots, phase plots, 2D curve fitting, state animation, FFT, histograms, correlation, heatmaps and data tools.
- View/session saving and complete-project ZIP saving for in-memory files.

## Requirements and limitations

- Windows 10 or Windows 11, x64.
- macOS Intel or Apple silicon.
- Linux x86_64.
- No Internet connection is required after installation.
- Complete-project ZIP export is intentionally unavailable while any loaded source is using lazy loading.
- Automatic updates are not included in this beta. Download newer releases manually from GitHub.

Please report reproducible problems in the [GitHub issue tracker](https://github.com/ferrucci-franco/timeseries-explorer/issues).
