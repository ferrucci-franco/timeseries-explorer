OpenModelica Viewer — Performance Benchmark

Imports CsvParser and PlotManager from src/ directly, so timings reflect the production code path. Run from npm run dev (Vite serves ESM correctly). Do not open this via file:// — the ES module imports will 404.

How to use
  1. Pre-generate synthetic CSVs (optional but recommended): 
    node bench/generate-synthetic-csv.mjs --rows 1500000 --out bench/data/synth-100mb.csv
node bench/generate-synthetic-csv.mjs --rows 15000000 --out bench/data/synth-1gb.csv
  2. Open this page via npm run dev (default http://127.0.0.1:8000/bench/benchmark.html).
  3. Select one or more CSV files. The runner parses each, renders a 1D plot, exercises zoom levels, and finishes with synthetic heatmaps.
  4. Copy the JSON result block at the end into bench/baseline.md (or bench/final.md).

For the heap-memory column to populate, run Chrome with --enable-precise-memory-info (other browsers will show null).
