Scenario A: xR=[-10,10], yR=[-1,1], zR=[-100,100] — non-cubic box shapes

Each plot uses aspectmode:'manual' with a different aspectratio so the rendered box is not a cube. The 3 axis lines must still appear equal on-screen.
Formula: L_i = C × span_i / ar_i where ar_i is the aspectratio for axis i. We cap C so every line endpoint stays inside its range.
